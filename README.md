# sunrce_go
go语言编写的并发向日葵rce检测、利用工具

![img](https://github.com/NewBeginning6/sunrce_go/blob/main/use.png)
**Usage:**

    -c string
        需要执行的命令，默认whoami (default "whoami")
    -p string
        端口号范围,默认40000-50000 例如:-p=80,81,88-1000 (default "40000-60000")
    -r string
        url list file
    -u string
        ip or url
